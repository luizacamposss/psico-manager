# PsicoManager 🧠

Sistema web de gerenciamento para clínicas psicológicas desenvolvido com Node.js, Express, MySQL e interface web responsiva.

---

# 📌 Objetivo

O PsicoManager foi criado para auxiliar psicólogos e clínicas no gerenciamento de:
- pacientes
- agendamentos
- prontuários
- financeiro
- tarefas
- dashboard administrativo

O sistema centraliza as informações da clínica em um único ambiente, facilitando a organização e acompanhamento dos atendimentos.
